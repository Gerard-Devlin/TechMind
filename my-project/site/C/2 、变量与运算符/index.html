    优先级表格

    目：参与一次运算的算子数

    Info

    单目和赋值是自右向左

    优先级 运算符 运算 结合关系 举例
    1 + 单目不变 自右向左 a*+b
    1 - 单目取负 自右向左 a*-b
    2 * 乘 自左向右 a*b
    2 / 除 自左向右 a/b
    2 % 取余 自左向右 a%b
    3 + 加 自左向右 a+b
    3 - 减 自左向右 a-b
    4 = 赋值 自右向左 a=b

    Note

    赋值也是运算，也有结果

    a=b=6 > a=(b=6)

    Warning

    “嵌入式赋值”不要用

    前缀/后缀

    表达式 运算 表达式的值
    count++ 给count加1 count原来的值
    ++count 给count加1 count+1以后的值
    count-- 给count减1 count原来的值
    --count 给count减1 count-1以后的值

    int main() {
    int a;
    a=10;
    printf("a++=%d\n",a++);
    printf("a=%d\n",a);

    ++a;
    printf("++a=%d\n",a);
    printf("a=%d\n",a);

    return 0;
}
